# SeniorConnect

Simple, large-text social app designed for seniors. Easy video calls with grandkids, local event discovery, medication reminders, and emergency contacts.

## Features

- **Profile**: Create and customize your profile
- **Feed**: Browse content from connections
- **Messaging**: Direct messaging with connections
- **Discovery**: Find new people and content

## Getting Started

```bash
# Install dependencies
npm install

# Start the development server
npm start

# Run on iOS
npm run ios

# Run on Android
npm run android
```

## Tech Stack

- **Framework**: React Native with Expo SDK 54
- **Language**: TypeScript
- **Navigation**: Expo Router
- **Styling**: NativeWind (Tailwind CSS)
- **State Management**: Zustand
- **API Client**: Axios with React Query
